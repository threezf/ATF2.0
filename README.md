# Vue-webpack-basic-fe

> 种子工程
>> fushoudong本地库
## 构建及运行

``` bash
# 安装
npm install

# 启动服务 localhost:8080
npm run dev

# 打包
npm run build
```

## 目录结构

#### src/components
放置所有的公用组件

#### src/const
放置一些常量

#### src/libs
公共函数

#### src/router
路由文件，目前是主路由和子页面的分路由

#### src/styles
样式文件

#### src/views
页面文件，按功能模块分
